## Run Locally

Clone the project

```bash
  git clone git@github.com:TVYDev/falcon-mart.git
```

Go to the project directory

```bash
  cd falcon-mart
```

Install dependencies

```bash
  yarn install
```

Start the development server

```bash
  yarn dev
```

Run test

```bash
  yarn test
```

## Run In Production

Build the project for production

```bash
  yarn build
```

Run the project

```bash
  yarn start
```
